## ما هي لعبة عائلة فيود؟

عائلة فيود لعبة بين فريقين. كل جولة يُطرح سؤال مثل:

> **"سمّ شيئاً تجده في المطبخ"**

الإجابات مخفية على الشاشة الكبيرة (التلفزيون). يتنافس الفريقان للحصول على أكبر عدد من النقاط بالإجابة على ما يراه المسؤولون عن الاستطلاع كأكثر الإجابات شيوعاً.

## إعداد اللعبة

### الخطوة 1 — المضيف يفتح اللعبة

يذهب المضيف إلى الصفحة الرئيسية ويختار **"عائلة فيود"**، ثم يضغط على **"استضافة"**.

### الخطوة 2 — إعداد الفريقين

تظهر صفحة الإعداد بالخيارات التالية:

| الإعداد | الخيارات | الافتراضي |
|---|---|---|
| اسم الفريق الأول | أي نص | الفريق الأول |
| اسم الفريق الثاني | أي نص | الفريق الثاني |
| أقصى عدد لاعبين لكل فريق | 2 — 6 أو غير محدود | 4 |
| عدد الجولات | 5 / 8 / 10 / 12 / 15 | 10 |

**مثال على إعداد مناسب:**
```
الفريق الأول: الأسود
الفريق الثاني: الأبيض
أقصى لاعبين: 4
عدد الجولات: 10
```

### الخطوة 3 — الحصول على رمز الغرفة

بعد الضغط على **"إنشاء الغرفة"** يُنقل المضيف تلقائياً إلى **غرفة الانتظار** ويظهر **رمز مكوّن من 4 أحرف** مثل `H7KR`.

المضيف يعرض هذا الرمز على الشاشة الكبيرة حتى يتمكن اللاعبون من الانضمام.

## انضمام اللاعبين

### كيف ينضم اللاعب؟

1. يفتح اللاعب اللعبة على **هاتفه** (الهاتف يعمل كوحدة التحكم).
2. يختار **"انضمام"** ويُدخل اسمه ورمز الغرفة.
3. تظهر له صفحة الفريقين مع عدد اللاعبين المتاحين:

```
┌─────────────────┐     ┌─────────────────┐
│   الأسود        │     │   الأبيض        │
│  ● أحمد         │     │  ● سارة         │
│  ● محمد         │     │  ○ فارغ         │
│  ○ فارغ         │     │  ○ فارغ         │
│  ○ فارغ         │     │  ○ فارغ         │
│  [انضم]  2/4    │     │  [انضم]  1/4    │
└─────────────────┘     └─────────────────┘
```

4. يضغط **"انضم"** على الفريق الذي يريده.
5. إذا كان الفريق ممتلئاً يظهر تنبيه ويطلب اختيار الفريق الآخر.

### ملاحظة للمضيف في غرفة الانتظار

يرى المضيف في غرفة الانتظار:
- قائمة اللاعبين مرتبة بحسب فريقهم.
- أيقونة تحذير إذا كان عدد لاعبي الفريقين غير متساوٍ.
- زر **"إعدادات"** لتعديل أسماء الفرق أو عدد الجولات قبل البدء.

بمجرد اكتمال اللاعبين يضغط المضيف **"ابدأ اللعبة"**.

## مراحل اللعبة

### نظرة عامة على تسلسل الجولة

```
[دوّارة الأسئلة]
       ↓
 [المواجهة]  ← كلا الفريقين يتنافسان
       ↓
[دور الفريق الفائز] ← يجيبون على الإجابات المتبقية
       ↓
   إما: [كل الإجابات ظهرت] → نهاية الجولة
   أو:  [3 أخطاء]          → [مرحلة السرقة]
                                     ↓
                             [نهاية الجولة]
       ↓
  [الجولة التالية] أو [نهاية اللعبة]
```

---

### المرحلة 1 — دوّارة الأسئلة 🎡

بين كل جولة وأخرى تظهر **دوّارة دوّارة** بألوان الفريقين على الشاشة الكبيرة.
الدوارة تدور لمدة **2.5 ثانية** ثم تختفي لتبدأ مرحلة المواجهة.

> الهدف: إضافة إثارة وإتاحة الوقت للاستعداد.

---

### المرحلة 2 — المواجهة ⚡

**السؤال يظهر على الشاشة الكبيرة. الإجابات مقلوبة.**

- **كلا الفريقين** يمكنهم الإجابة في نفس الوقت — لا يوجد دور محدد.
- **أسرع لاعب يُجيب بشكل صحيح** يفوز بالمواجهة.
- إذا كانت الإجابة **خاطئة** لا توجد عقوبة — الفرصة لا تزال متاحة للجميع.
- يستمر المضيف في المواجهة يدوياً إذا لم يجب أحد صحيحاً (زر "تخطي المواجهة").

**على هاتف اللاعع يظهر:**
```
┌───────────────────────────┐
│ ⚡ المواجهة               │
│ أسرع في الإجابة!          │
│                           │
│ ما هو أكثر لون تحبه؟      │
│                           │
│ [ اكتب إجابتك...    ] [→] │
└───────────────────────────┘
```

**الفريق الفائز بالمواجهة** يحصل على:
- نقاط الإجابة الأولى.
- **السيطرة على اللوحة** في المرحلة التالية.

---

### المرحلة 3 — دور الفريق 🟢

الفريق الفائز بالمواجهة يتحكم باللوحة.

- **فقط لاعبو الفريق النشط** يمكنهم الإجابة.
- **كل لاعع له محاولة واحدة فقط** في كل جولة.
- الإجابات الصحيحة تُكشف على الشاشة مع صوت "كشف".
- الإجابات الخاطئة تُضيف **خطأ (❌)** لعداد الفريق.
- بعد **3 أخطاء** ينتهي الدور ويبدأ **وقت السرقة**.
- يمكن للاعع أن يضغط **"تمرير الدور للسرقة"** إذا أراد الانتهاء مبكراً.

**على هاتف لاعع الفريق النشط:**
```
┌───────────────────────────┐
│ أحمد — الأسود             │
│ ✅ دور فريقكم!            │
│    اكتب إجابتك!           │
│                           │
│ ما هو أكثر لون تحبه؟      │
│                           │
│ [ اكتب إجابتك...    ] [→] │
│                           │
│ [تمرير الدور للسرقة →]    │
└───────────────────────────┘
```

**على هاتف لاعع الفريق المنتظر:**
```
┌───────────────────────────┐
│ سارة — الأبيض             │
│ ⏳ دور الفريق الآخر...    │
│    الأسود يلعب الآن       │
└───────────────────────────┘
```

---

### المرحلة 4 — مرحلة السرقة ⚡🔴

تبدأ عندما يُخطئ الفريق النشط **3 مرات** أو يمرر دوره.

**قواعد السرقة:**
- **فريق السرقة** هو الفريق الآخر (المنتظر).
- **فقط القائد** (أول لاعع انضم لفريق السرقة) يمكنه الإجابة.
- **محاولة واحدة فقط** — إما كل شيء أو لا شيء.
- يمكن لبقية لاععي الفريق **مناقشة الإجابة** مع قائدهم قبل أن يرسلها.

**على هاتف القائد:**
```
┌───────────────────────────┐
│ محمد — الأبيض             │
│ ⚡ أنت القائد!            │
│ فرصة السرقة               │
│ اكتب إجابة فريقك          │
│                           │
│ [ اكتب إجابتك...    ] [→] │
└───────────────────────────┘
```

**على هاتف بقية لاععي فريق السرقة:**
```
┌───────────────────────────┐
│ 🤝 قائد فريقك يجاوب الآن │
│ ناقشوا الإجابة معاً       │
└───────────────────────────┘
```

**على هاتف لاععي الفريق المحجوب:**
```
┌───────────────────────────┐
│ ❌ الفريق الآخر يحاول     │
│    السرقة                 │
└───────────────────────────┘
```

## طريقة الإجابة

### من يمكنه الإجابة متى؟

| المرحلة | من يمكنه الإجابة |
|---|---|
| المواجهة | **جميع اللاععين** من كلا الفريقين |
| دور الفريق | **فقط لاععو الفريق النشط** (كل واحد مرة واحدة) |
| السرقة | **فقط القائد** (أول لاعع انضم لفريق السرقة) |

### كيف تُحسب الإجابة صحيحة؟

النظام يقبل الإجابات بمرونة عالية:
- يتجاهل الحركات والتشكيل (مثلاً: "أزرق" و"أَزْرَق" نفسهما).
- يتجاهل "ال" التعريف (مثلاً: "السيارة" و"سيارة" نفسهما).
- يقبل التهجئة المتقاربة (مثلاً: "تفاحه" و"تفاحة" نفسهما).
- يُطابق الكلمات المشابهة (أخطاء إملائية بسيطة مقبولة).

## نظام النقاط

### كيف تُحسب النقاط في كل جولة؟

كل إجابة على اللوحة لها **قيمة نقاطية** حسب شيوعها في الاستطلاع.

```
مثال: "سمّ لوناً أساسياً"
┌──────────────────────────┐
│ 1. أحمر         ← 42 نقطة │
│ 2. أزرق         ← 28 نقطة │
│ 3. أصفر         ← 18 نقطة │
│ 4. أخضر         ← 12 نقطة │
└──────────────────────────┘
```

**تراكم النقاط في الجولة (roundScore):**
- كل إجابة صحيحة **تُضاف** إلى رصيد الجولة.
- الرصيد يتراكم حتى نهاية الجولة ثم يُعطى للفريق الفائز.

### مَن يحصل على نقاط الجولة؟

| الحالة | مَن يحصل على النقاط |
|---|---|
| الفريق النشط يكشف كل الإجابات | الفريق النشط يحصل على كامل الرصيد |
| سرقة ناجحة ✓ | فريق السرقة يحصل على **رصيد الجولة + نقاط الإجابة المسروقة** |
| سرقة فاشلة ✗ | الفريق النشط الأصلي يحصل على **رصيد الجولة** كما هو |

**مثال عملي:**
```
الجولة 5:
  الفريق الأسود يكسب المواجهة
  يكشف: أحمر (42) + أزرق (28) = رصيد الجولة: 70 نقطة
  يُخطئ 3 مرات → تبدأ مرحلة السرقة

  الفريق الأبيض يحاول السرقة:
    ✓ إذا أجاب "أصفر" (18) → يحصل على 70 + 18 = 88 نقطة
    ✗ إذا أخطأ            → الفريق الأسود يحصل على 70 نقطة
```

## أدوات المضيف

في الجزء السفلي من الشاشة الكبيرة يوجد **لوح تحكم للمضيف** بالأدوات التالية:

### الصف الأول — التحكم في سير اللعبة

| الزر | الوظيفة |
|---|---|
| ⏭️ التالي / تخطي المواجهة | الانتقال للسؤال التالي (من نهاية الجولة)، أو تخطي مرحلة المواجهة وإعطاء الدور لفريق 1 |
| 🔄 إعادة السؤال | إعادة ضبط السؤال الحالي من الصفر (تُخفى كل الإجابات، تُصفّر الأخطاء) |
| ⏸️ إيقاف مؤقت | تجميد اللعبة — لا يمكن لأحد الإجابة حتى يضغط المضيف "استمرار" |
| 🏁 إنهاء اللعبة | إنهاء اللعبة فوراً والانتقال لشاشة الفائز |

### الصف الثاني — التحكم في اللوحة

| الزر | الوظيفة |
|---|---|
| ↩️ تراجع | إخفاء آخر إجابة ظهرت وإعادة نقاطها من الرصيد |
| ✗ خطأ يدوي | إضافة خطأ للفريق النشط (مفيد عند الخطأ الشفهي) |
| 🔢 تعديل النقاط | نافذة لتعديل نقاط كلا الفريقين مباشرة |
| 🎯 القفز لسؤال | اختيار أي سؤال من قائمة للقفز إليه مباشرة |

### الصف الثالث — كشف الإجابات يدوياً

لكل إجابة **لم تُكشف بعد** يظهر زر يحمل رقمها. يضغط المضيف عليه ليكشف الإجابة ويضيف نقاطها للرصيد.

**مثال:**
```
[كشف 3] [كشف 4] [كشف 5]
```

## الفوز بالمباراة

### متى تنتهي اللعبة؟

تنتهي اللعبة عندما:
- تنتهي **جميع الجولات** المحددة مسبقاً، أو
- يضغط المضيف **"إنهاء اللعبة"** يدوياً.

### مَن يفوز؟

الفريق الذي يمتلك **أعلى مجموع نقاط** عند نهاية اللعبة يفوز.

تظهر شاشة الفائز مع مجموع نقاط كلا الفريقين.

## مثال كامل على جولة

لنفترض أن الجولة 3 قد بدأت بالسؤال:

> **"سمّ وسيلة مواصلات"**

```
الإجابات على اللوحة (مخفية):
┌──────────────────────────────┐
│ ■■■■■■■■      44 نقطة       │
│ ■■■■■■■■      31 نقطة       │
│ ■■■■■■■■      15 نقطة       │
│ ■■■■■■■■       6 نقطة       │
│ ■■■■■■■■       4 نقطة       │
└──────────────────────────────┘
```

**المواجهة:**
- أحمد (الأسود) يكتب "سيارة" ← ✓ إجابة صحيحة! (44 نقطة)
- يُكشف المربع الأول: **سيارة — 44 نقطة**
- رصيد الجولة: **44**
- الفريق الأسود يحصل على السيطرة

**دور الفريق الأسود:**
- محمد يكتب "طيارة" ← ✓ صحيح! (31 نقطة) — رصيد: **75**
- فاطمة تكتب "بحر" ← ✗ خطأ! — عداد الأخطاء: **1**
- خالد يكتب "قطار" ← ✓ صحيح! (15 نقطة) — رصيد: **90**
- سلمى تكتب "موتور" ← ✗ خطأ! — عداد الأخطاء: **2**
- *(لا يوجد لاعع خامس)* — الفريق استنفد لاععيه → يمرر الدور تلقائياً إلى السرقة

**مرحلة السرقة (الفريق الأبيض):**
- سارة هي القائدة (أول من انضم للفريق الأبيض)
- تتشاور مع فريقها، تقرر الإجابة بـ"باص"
- تكتب "باص" ← ✓ إجابة صحيحة! (6 نقطة)

**نتيجة الجولة:**
```
الفريق الأبيض يسرق الجولة!
يحصل على: 90 (رصيد) + 6 (الإجابة المسروقة) = 96 نقطة
الإجابة الباقية (دراجة - 4 نقاط) لم تُكشف.
```

**اللوحة النهائية:**
```
✓ سيارة        44
✓ طيارة        31
✓ قطار         15
✓ باص           6     ← كشفتها سارة بالسرقة
■ دراجة         4     ← لم تُكشف
```

## ملاحظات للمضيف

- **ابدأ بإحماء**: خصص الجولات الأولى للأسئلة السهلة.
- **استخدم الإيقاف المؤقت**: عند أي مشكلة تقنية أو فاصل.
- **تراجع عن الأخطاء**: إذا أضفت خطأً بالغلط استخدم زر ↩️ تراجع.
- **كشف يدوي**: إذا أعطى لاعع إجابة صحيحة شفهياً ولم يقبلها النظام، اكشف الإجابة يدوياً من الزر المناسب.
- **تحذير الفرق غير المتوازنة**: إذا كان عدد لاععي الفريقين غير متساوٍ، يظهر تحذير في غرفة الانتظار قبل البدء.
